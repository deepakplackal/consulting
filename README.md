# Antriom Advisory Website

A professional, single-page marketing site for **Antriom Advisory**, a strategic financial advisory firm focused on MSME restructuring, capital strategy, and continuity planning.

## What’s inside

- `index.html` — content and structure
- `styles.css` — brand styling (deep green + gold palette)
- `script.js` — mobile menu + form UX
- `scripts/validate.js` — quick validation check

## Run locally

This is a static site. Start a local server and open the site in your browser.

```bash
npm run start
```

Then open `http://localhost:8000`.

## Validate files

```bash
npm test
```

## Customize

- Update contact details in `index.html`.
- Swap the SVG logo for a final brand asset when ready.
- Add a lead capture endpoint in `script.js` when you integrate a backend.
